    Cycle 1
    1. 초기 로드 시 real DOM을 읽어 test 영역에 복제합니다.
    2. test 영역의 텍스트, 속성, 자식 구조를 직접 수정합니다.
    3. `Patch`를 눌러 real 영역에 patch를 적용합니다.
    4. `Undo`, `Redo`로 history snapshot 이동을 확인합니다.
